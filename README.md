📱 Sobre o projeto

O Finanças é um app web responsivo para controle de gastos pessoais ou compartilhados. Basta criar um código de sala, entrar com seu nome e começar a lançar receitas e despesas — sem cadastro, sem complicação.

Dois ou mais usuários no mesmo código de sala veem os dados atualizados em tempo real, ideal para casais, roommates, amigos ou qualquer pessoa que queira dividir o controle financeiro.


✨ Funcionalidades


🔐 Acesso por código de sala — sem login ou senha
⚡ Sincronização em tempo real via Firebase Firestore
💸 Lançamento de receitas e despesas com categorias e emojis
✓ Marcação de despesas como quitadas
📊 Gráficos de despesas por categoria (donut) e histórico de 6 meses (barras)
🎯 Sistema de metas com barra de progresso
📅 Filtro por mês
📱 Layout mobile-first, funciona como PWA



🛠️ Tecnologias


HTML, CSS e JavaScript puro (sem frameworks)
Firebase Firestore para banco de dados em tempo real
Firebase SDK via CDN (ES Modules)
